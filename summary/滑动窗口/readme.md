## LeetCode分类刷题之滑动窗口(Sliding window)

滑动窗口经常是用来执行数组或是链表上某个区间（窗口）上的操作。
比如找最长的全为1的子数组长度。滑动窗口一般从第一个元素开始，一直往右边一个一个元素挪动。
